# 🚢 NORSEMAN YACHTS 官网

---

## ⚡ 三步启动

### 1️⃣ 安装依赖
```bash
npm install
```

### 2️⃣ 启动项目
```bash
npm start
```

### 3️⃣ 访问网站
浏览器会自动打开 **http://localhost:3000** 🎉
